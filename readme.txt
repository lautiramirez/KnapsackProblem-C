Instrucciones para compilar:

    Utilizando el archivo Makefile, ingresando en la terminal el siguiente comando:
        $ make all

Instrucciones para ejecutar el programa:

    Ingresar en su terminal el siguiente comando:
        $ ./knapsack -b -f input/example1.in -t (BACKTRACKING).
        $ ./knapsack -d -f input/example1.in -t (DINAMICA).
        $ ./knapsack -ds -f input/example1.in -t (DINAMIC SELECTION).

        NOTA: Remplazar "example1.in" por el archivo que desee probar.
	
NOTA: Para borrar los archivos generados simplemente ingrese en su terminal el comando "make clean".
NOTA: Para que el programa corra, desde la terminal hay que ir al directorio donde se encuentra el archivo.
